# Jarvis - First Steps

## Description

Jarvis is a personal assistant that can help you with your daily tasks.

## Installation

1 - Setup OpenClaw

    Linux: 

    (Bash)
    curl -fsSL https://openclaw.ai/install.sh | bash


    Windows: 

    (PowerShell)
    iwr -useb https://openclaw.ai/install.ps1 | iex 


2 - Say yes to ask about risks

3 - Select your LLM model, here we will use OpenRouter

    Create account: https://openrouter.ai/

    Generate key : https://openrouter.ai/settings/keys

    Get a model: https://openrouter.ai/models
